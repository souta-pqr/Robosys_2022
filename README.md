# plusコマンド

![test](https://github.com/souta-pqr/Souta-Kobori/actions/workflows/test.yml/badge.svg)

## 何をするためのソフトか
標準入力から読み込んだ数字を足すためのもの.

## 必要なソフトウェア
* python
  * テスト済み: 3.7～3.10

## テスト環境
* ubuntu 18.04

## 起動する手順
* ubuntu 18.04のターミナル上で、`git clone <リポジトリのURL>` をした後`cd <リポジトリ名>`で移動する。

## 簡単な使い方
* `seq 5 | ./plus`
  * 実行結果→15
